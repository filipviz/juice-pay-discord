# juice-pay-discord

Send Discord Webhook notifications when any [Juicebox project](https://juicebox.money) receives a payment.

## How to Run

1. Create a .env file:

```bash
cp .example.env .env
```

2. Fill out your .env file.

3. Install [node.js](https://nodejs.org/) (>=18.0.0), and run with:

```bash
node .
```
